# Next.js Payload App

This is a Next.js application that uses Payload CMS for content management. The app is configured to use SQLite as the database.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000/admin](http://localhost:3000/admin) with your browser and login as with your admin credential to set up the CMS.
Open [http://localhost:3000/home](http://localhost:3000/home) with your browser to see the result.
